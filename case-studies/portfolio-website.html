Project Overview

This portfolio website rebuild transformed a single-page skeuomorphic design into a modern multi-page architecture with a dark blueprint aesthetic. The redesign focused on improving content organization, removing unused features (homelab stats dashboard and contact form), and modernizing the visual language while preserving all personal information and professional credentials.

The site is built with pure HTML, Tailwind CSS via CDN, and vanilla JavaScript—no build process required. This approach ensures fast load times, easy deployment, and minimal maintenance overhead.

Design System

The dark blueprint aesthetic uses electric blue (#135bec) as the primary accent color against a dark background (#101622). The design language draws inspiration from technical blueprints and cyberpunk aesthetics with glass morphism effects, grid patterns, and monospace typography for technical elements. 

/* Core Design Tokens */
--primary: #135bec;           /* Electric Blue */
--background-dark: #101622;   /* Deep Navy */
--background-light: #f6f6f8;  /* Light Gray */

/* Typography */
font-family: 'Space Grotesk', sans-serif;

/* Glass Morphism Effect */
.glass {
  background: rgba(19, 91, 236, 0.05);
  backdrop-filter: blur(12px);
  border: 1px solid rgba(19, 91, 236, 0.2);
}

Space Grotesk font provides a modern, geometric feel for display text while Material Icons supply consistent iconography throughout the interface.

Architecture & Navigation

The site uses a three-page architecture:

  • index.html: Main portfolio page with hero section, credentials badges, 6 skill cards, 3 featured projects, embedded resume PDF, and contact information.
  • projects.html: Projects listing page with filter/sort toolbar UI and project cards linking to detailed case studies.
  • case-studies/*.html: Individual project case studies with detailed technical breakdowns, metrics, and architecture explanations.

Fixed navigation header provides consistent access to Home, Projects, Resume, and Schedule Call CTA across all pages. Mobile navigation uses a slide-in drawer menu with smooth transitions.

Content Migration

All content from the original portfolio was carefully migrated and reorganized:

  • Personal Information: Name, location, email, phone, LinkedIn, GitHub, Cal.com scheduling link preserved and enhanced with card-based layout.
  • Credentials: Three associate degrees (Network Administration, Cybersecurity, Mathematics) displayed as prominent badges in hero section.
  • Skills: Six skill categories with technology tags migrated from original layout to modern card grid with hover effects.
  • Resume: PDF embedded with iframe and download link maintained, moved to organized assets directory.
  • Projects: Homelab infrastructure content expanded into detailed case study with technical deep-dive sections.

Features Removed

Several features were intentionally removed to streamline the portfolio:

  • Homelab Stats Dashboard: Live stats integration removed (homelab-stats.json and collect-homelab-stats.py) as the information is better served in the detailed case study format.
  • Contact Form: Google Apps Script backend removed in favor of direct links to email, LinkedIn, and Cal.com scheduling.
  • Skeuomorphic Design System: Entire custom CSS design system (skeuomorphic-design-system.css) replaced with Tailwind CSS utility classes.

Performance & Optimization

Performance optimizations include:

  • Zero Build Process: No npm dependencies, no bundling, no transpilation—just static HTML/CSS/JS served directly.
  • CDN Delivery: Tailwind CSS and fonts loaded from CDN with proper caching headers.
  • Lazy Loading: Images and iframe content use native lazy loading attributes.
  • Minimal JavaScript: Only essential JavaScript for mobile menu toggle—no heavy frameworks or libraries.
  • Responsive Design: Mobile-first approach with breakpoints at 768px (md) and 1024px (lg) for optimal viewing on all devices.

Lessons Learned

  • Content First: Starting with content organization and migration ensures no information is lost during redesign.
  • Design Systems: Using a mature design system (Tailwind CSS) accelerates development and ensures consistency.
  • Simplicity Wins: Removing unused features and complexity makes the site easier to maintain and faster to load.
  • Multi-Page Benefits: Breaking content into logical pages improves SEO and allows visitors to focus on specific areas of interest.
  • Template Reuse: Creating reusable templates for case studies enables rapid expansion of portfolio content.

Future Enhancements

  • Blog Functionality: Add markdown-based blog system for technical writing and project updates.
  • Project Filtering: Implement functional filter/sort on projects page by category, technology, or date.
  • Custom Diagrams: Create SVG architecture diagrams for case studies to visualize system designs.
  • Dark/Light Toggle: Add theme switcher for users who prefer light mode.
  • Analytics Dashboard: Create internal analytics dashboard to track portfolio performance metrics.
